# PyTools

A collection of minimalist Python utilities for daily development.

---

## 🌲 Tree Generator
Generates a clean ASCII directory tree structure for documentation. 
It automatically ignores junk folders like `.git` and `node_modules`. 
Directories are sorted above files for better readability. 
Usage: `python tree_generator.py [target_path]`. 
This tool is zero-dependency and uses only standard libraries.
